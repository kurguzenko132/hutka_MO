# Исправление после первого деплоя

## Что исправлено

1. Убрали жесткий редирект с `/` на `/dashboard`.
2. Добавили нормальную стартовую страницу `/` с презентацией Hutka и кнопками перехода.
3. Добавили красивую страницу `404`, чтобы вместо пустой ошибки был понятный экран.
4. Переименовали часть метрик на главной:
   - «Всего лидов» → «Всего контактов»
   - «Готовы тестировать» → «Готовы к пилоту»
   - «Активные тестеры» → «Активные участники»
   - «Просрочено задач» → «Просроченные действия»
5. Обновили кнопку в верхней панели: «Добавить контакт».

## Что сделать после замены файлов

```bash
git add .
git commit -m "Fix root page and improve MVP navigation"
git push
```

После push Vercel автоматически пересоберет проект.

## Какие ссылки проверить после деплоя

