# STEP 16 — Настройки справочников

В этом шаге добавлен рабочий раздел `/settings`.

## Что появилось

- базовые настройки приложения:
  - название продукта;
  - рабочее пространство;
  - город по умолчанию;
  - день недельного отчета;
- управление источниками контактов;
- управление стадиями воронки;
- управление тегами;
- редактирование цвета, типа и порядка;
- счетчик использований справочника;
- удаление справочников, если они не используются в связанных контактах;
- fallback на демо-данные, если Supabase не настроен.

## Что заменить

Главные файлы:

```text
lib/settings.ts
actions/settings.actions.ts
app/(dashboard)/settings/page.tsx
supabase/schema.sql
STEP16_SETTINGS_DIRECTORIES.md
```

## Обновление Supabase

Открой Supabase → SQL Editor и выполни обновленный файл:

```text
supabase/schema.sql
```

В этом шаге добавлена таблица:

```text
app_settings
```

И индексы для справочников:

```text
sources_name_idx
sources_type_idx
funnel_stages_type_order_idx
tags_name_idx
```

## Проверка

Открой:

```text
/settings
```

Проверь сценарии:

1. Измени название рабочего пространства.
2. Добавь новый источник, например `Instagram Reels`.
3. Добавь новую стадию, например `Презентация`.
4. Добавь новый тег, например `Нужно дожать`.
5. Открой `/people/new` и проверь, что новые значения можно использовать при создании контакта.
6. Открой `/funnels` и проверь, что новая стадия появилась в воронке.
