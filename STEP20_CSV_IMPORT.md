# STEP 20 — CSV импорт контактов

## Что добавлено

В этом шаге добавлен массовый импорт контактов из CSV-файла:

- страница `/people/import`;
- загрузка CSV-файла через server action;
- поддержка колонок на английском и русском;
- автоматическое создание источников, стадий и тегов, которых еще нет;
- пропуск дублей по Instagram, Telegram, телефону или email;
- запись истории импорта в карточку каждого контакта;
- журнал последних импортов;
- скачивание шаблона CSV: `/people/import/template`;
- таблица `import_logs` в Supabase.

## Поддерживаемые колонки

```text
name,type,niche,city,instagram,telegram,phone,email,source,stage,priority,tags,next_step,next_contact_date,notes
```

Также можно использовать русские заголовки:

```text
имя,тип,ниша,город,инстаграм,телеграм,телефон,почта,источник,стадия,приоритет,теги,следующий шаг,следующий контакт,заметки
```

## Как проверить

1. Обнови Supabase schema.sql.
2. Зайди в `/people/import`.
3. Скачай шаблон CSV.
4. Заполни 2–3 строки.
5. Загрузи файл.
6. Открой `/people` и проверь, что контакты появились.
7. Открой карточку импортированного контакта и проверь историю.
