# STEP 21 — Массовые действия в разделе «Люди»

## Что добавлено

На этом этапе раздел `/people` стал удобнее для работы с большими базами контактов после CSV-импорта.

Добавлено:

- выбор нескольких контактов в таблице;
- выбор всех контактов на текущей странице;
- массовая смена стадии;
- массовое добавление тега;
- массовое создание задачи для выбранных контактов;
- массовое добавление контактов в кампанию;
- запись массовых действий в историю каждого выбранного контакта;
- пересчет связанных разделов после массового действия:
  - `/people`;
  - `/funnels`;
  - `/dashboard`;
  - `/reports`;
  - `/geography`;
  - `/tasks`;
  - `/campaigns`.

## Зачем это нужно

Когда ты импортируешь 50–300 мастеров из CSV, работать по одному контакту неудобно. Теперь можно:

- выбрать всех мастеров из конкретного города;
- назначить им тег `Минск / маникюр`;
- перевести их в стадию `Написал`;
- создать задачу `Отправить первое сообщение`;
- добавить всех в кампанию `Мастера маникюра Минск — Instagram`.

## Проверка

1. Открой `/people`.
2. Выбери несколько контактов чекбоксами.
3. Вверху появится панель **«Массовые действия»**.
4. Проверь смену стадии.
5. Проверь добавление тега.
6. Проверь создание задачи для выбранных контактов.
7. Проверь добавление выбранных контактов в кампанию.
8. Открой карточку одного контакта и проверь, что в истории появилась запись о массовом действии.
