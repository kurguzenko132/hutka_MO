# STEP 31 — Готовые паки вопросов для контактов

## Что добавлено

В карточке контакта появился блок **«Готовые паки вопросов»**. Теперь маркетологу не нужно каждый раз вручную писать вопросы для мастера, салона или клиента.

Можно нажать **«Создать ссылку»** у готового пака — Hutka сразу создаст персональную анкету `/q/[token]` с нужными вопросами и привяжет ее к текущему контакту.

## Доступные паки

1. **Мастер: диагностика**  
   Базовый пакет для индивидуального мастера: ниша, город, текущая запись, боли, CRM, готовность к пилоту.

2. **Мастер: профиль на карте**  
   Пак для сбора данных для карточки мастера на карте: услуги, цены, описание, график, формат приема, материалы.

3. **Салон: диагностика**  
   Пак для салонов: команда, запись, CRM, администраторы, роли, пустые окна, интерес к карте.

4. **Фидбек после теста**  
   Пак после пилота: что понятно, что неудобно, оценка, недостающие функции, готовность пользоваться дальше.

5. **Причина отказа**  
   Короткий пакет, чтобы понять, почему контакт отказался или поставил паузу.

6. **Клиент: карта**  
   Пак для B2C-клиентов: как ищут мастеров, что важно в карточке, готовы ли записываться через карту.

## Как пользоваться

1. Открой `/people`.
2. Перейди в карточку контакта.
3. Найди блок **«Готовые паки вопросов»**.
4. Нажми **«Создать ссылку»** у нужного пака.
5. Скопируй ссылку из блока **«Персональные анкеты»**.
6. Отправь ссылку человеку.
7. После ответа результаты появятся в карточке контакта.

## Проверка

Открой карточку любого контакта:

```text
/people/[id]
```

Проверь:

1. Есть блок **«Готовые паки вопросов»**.
2. У каждого пака есть описание и количество вопросов.
3. Нажатие **«Создать ссылку»** создает персональную анкету.
4. Новая анкета появляется в блоке **«Персональные анкеты»**.
5. Ссылка `/q/[token]` открывается без входа.
6. Ответы сохраняются в карточке контакта.
