# STEP 34 — Редактор шаблонов сообщений

## Что добавлено

Добавлен следующий рабочий модуль Hutka: **шаблоны сообщений**.

Теперь администратор может управлять готовыми текстами из интерфейса, а маркетолог может копировать персонализированное сообщение прямо из карточки контакта.

## Новые возможности

- Новый раздел:
  - `/settings/message-templates`
- Страница конкретного шаблона:
  - `/settings/message-templates/[id]`
- Создание шаблона сообщения.
- Редактирование шаблона сообщения.
- Удаление шаблона сообщения.
- Настройка аудитории:
  - мастер;
  - салон;
  - клиент;
  - партнер;
  - любой контакт.
- Настройка категории:
  - первое касание;
  - анкета;
  - follow-up;
  - пилот;
  - отказ / пауза;
  - фидбек;
  - другое.
- Настройка канала:
  - Instagram;
  - Telegram;
  - WhatsApp / Viber;
  - Email;
  - телефон;
  - любой канал.
- В карточке контакта появился блок **«Шаблоны сообщений»**.
- Шаблон автоматически подставляет данные контакта.
- Можно вставить ссылку на персональную анкету.
- Сообщение можно скопировать одной кнопкой.

## Поддерживаемые переменные

В тексте шаблона можно использовать:

```text
{{name}}
{{first_name}}
{{type}}
{{niche}}
{{city}}
{{stage}}
{{source}}
{{instagram}}
{{telegram}}
{{phone}}
{{email}}
{{questionnaire_link}}
{{app_name}}
{{sender_name}}
{{sender_title}}
{{sender_email}}
```

## Пример шаблона

```text
Привет, {{first_name}}! Мы сейчас запускаем Hutka — сервис, где beauty-мастера смогут получать заявки через карту и удобнее вести запись. Можно отправить короткую анкету?
```

## Supabase

Добавлена таблица:

```text
message_templates
```

Также добавлены RLS-политики:

- все авторизованные пользователи могут читать шаблоны;
- только `admin` может создавать, редактировать и удалять шаблоны.

## Что проверить

1. Открыть `/settings/message-templates` под admin.
2. Создать новый шаблон.
3. Открыть карточку контакта `/people/[id]`.
4. Найти блок **«Шаблоны сообщений»**.
5. Выбрать шаблон.
6. Вставить ссылку на анкету, если нужно.
7. Скопировать текст.
8. Проверить, что переменные заменились на данные контакта.
