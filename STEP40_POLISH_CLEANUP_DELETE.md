  - задача;
  - персональная анкета контакта;
  - опросник;
  - вопрос опросника;

  - гипотеза.
- В настройках добавлена карточка «Очистка базы».

## Важные замечания

Очистка через интерфейс использует `SUPABASE_SERVICE_ROLE_KEY`, потому что обычные RLS-права пользователя могут не дать удалить все данные пачкой.

Профили пользователей, Supabase Auth и `app_settings` не удаляются, чтобы не потерять доступ к админке.

## Как очистить данные

Открой:

```text
/settings/data-cleanup
```

Выбери режим:

```text
Только рабочие данные
```

или:

```text
Рабочие данные + справочники и шаблоны
```

В поле подтверждения напиши:

```text
ОЧИСТИТЬ
```

Если кнопка не сработает из-за прав, выполни в Supabase SQL Editor:

```text
supabase/cleanup-workspace.sql
```
