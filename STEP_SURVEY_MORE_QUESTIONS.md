# Fix: больше вопросов при создании анкеты

Что изменено:

- На странице `/surveys/new` блок вопросов стал динамическим.
- Теперь можно нажать **«Добавить вопрос»** и добавить больше 6 вопросов.
- Максимум в интерфейсе — 50 вопросов, чтобы случайно не перегрузить форму.
- Пустые вопросы не сохраняются.
- Лишние вопросы можно удалить прямо при создании анкеты.
- Server action больше не ограничен фиксированным циклом на 6–8 вопросов: он собирает все поля `question_text_N`, которые пришли из формы.

Файлы:

- `components/surveys/survey-question-builder.tsx`
- `app/(dashboard)/surveys/new/page.tsx`
- `actions/surveys.actions.ts`
